# HROne Backend Intern Task

## 🚀 Tech Stack
- FastAPI
- MongoDB (Motor)
- Python 3.10+

## ✅ API Endpoints

### POST `/products`
- Create a new product

### GET `/products`
- List products
- Filters: name, size
- Pagination: limit, offset

### POST `/orders`
- Create an order with productIds and qty

### GET `/orders/{user_id}`
- List orders for user
- Includes product details and total price
- Pagination
